# React + TypeScript + Vite

## 프로젝트 설정 순서

1. Yarn Berry 버전 세팅

```bash
yarn set version berry
```

2. 의존성 파일 설치

```bash
yarn
```

3. 서버 실행

```bash
yarn dev
```
